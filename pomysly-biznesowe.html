⚙️ Jeden raz: podaj dane Firebase

Wklej tutaj dane z Firebase żeby włączyć synchronizację z Grzesiem.

Jak to zrobić (raz na zawsze):
1. Wejdź na console.firebase.google.com
2. Utwórz projekt → Realtime Database → tryb testowy
3. Skopiuj URL bazy, klucz API i identyfikator projektu
4. Wklej powyżej i kliknij „Zapisz i połącz"
